落叶焚

发表于 2017-05-21 | 分类于 杂谈

渡的诗选

诗选

手机地图

我定格在
一枚小小的图标
从前，往后
左右不得
唯有收拢双指，徒劳为之

当你收拢一定程度
就能看到故乡
食指可触
我触摸着，回家的路
抚摸在华北平原的铁轨上
大半个中国
一路摸到
门前的山茶
放在我的心上

滑板行

我在陌生的城市
刷你走过的街

大地之舟
城市的起伏
可曾默数
你心海的涛声
我正在去
你曾去过的地方

我想起
远在千里外的故乡
滑过的每一寸故土
孤帆远影
脚下，是滚滚而去的大地

淘金者

此刻
你终于站在了这里
一个失去本名的河畔
淘金者的天堂

当你在
满是谎言的河流中
淘洗真理
可曾令你喜悦 或是悲哀
狂热席卷 弹指尘埃

终有一日
你练就一身水性
纵身投入河流
留下空洞的
竹篮

你沙制的城堡……

童年在海边堆砌的城堡
在大海的冲刷下消失殆尽
后来，我再也没堆过城堡
直至遗忘
也许某天猛然想起
我能堆砌城堡
我甚至不敢去想
曾经，我拥有一座城堡

南墙

残存的意识里
那个逐渐湮没的传闻
多少嗤之以鼻
而我深信不疑
谣言流传数年不攻自破
唯有真相传说千年

我已经 失去了很多
随着年长
我也很难 轻易相信什么
也许这也终将抹去

就让我
再撞一次南墙
没有了墙 你可以到达
世界的每一角落

麻雀

去图书馆路上
惊起几只麻雀
鸟鸣稍纵即逝
我几乎遇到了 所有的生灵

尘封的俄国望远镜
压着《中国鸟类野外手册》
一拖再拖
却又挥之不去

麻雀争恐地逃离着
明日重返的树梢
我在这里行走 奔跑
日复一日
走不出的雾霾
穷尽一生的力气
走不出的 莫比乌斯环

房间

下午，我在六楼
楼下足球赛
稀疏的哨声
球员的交谈与踢球
那么清晰可闻

风敲开了窗
我拉下门闩
想让风进来
他却为我掩上门

一缕阳光
打在俄国地图上
目随光斑的移动
明天我也会离去
最后一次凝望
高加索沐浴的光明

远古机械巨人

每当秋叶远去
我伸开双臂
燃烧着拥抱大地
北风凛然
热血流淌冰河

多少冬夜
铁拳抨击着胸膛
那冰封的铁炉
沉闷作响
隐隐透着微光

听，那是风箱
而我却成冰霜
